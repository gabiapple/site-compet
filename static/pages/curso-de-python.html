Curso de Python

O curso de linguagem Python oferecido pelo COMPET teve como objetivo a capacitação dos alunos na linguagem de programação Python, sendo portanto uma atividade de ensino. O curso envolveu os níveis básico e intermediário da linguagem. As aulas envolveram teoria e prática, sendo exemplificado e testado pelos alunos todos os conceitos propostos, à medida que eram passados. O curso foi organizado da seguinte maneira:

# Horário Atividade
1 24/10 - 13:00 as 17:00 Conceitos básicos da linguagem: tipos e variáveis, comandos de entrada e saída, estrutura de dados nativas, estruturas condicionais e de repetição.
2 01/10 - 13:00 as 17:00 Conceitos intermediários da linguagem: Funções. Exercícios de todos os tópicos. Tópicos extras.

A atividade foi realizada por conta de uma demanda do curso de Administração do CEFET-MG repassada pelo grupo PET de Administração. Houve ainda uma demanda de capacitação dos integrantes do COMPET, pois há utilização da linguagem Python para o desenvolvimento de websites dentro do próprio grupo. Considerando o quórum obtido, foi sugerido que se abrisse o curso para outros alunos do CEFET-MG, aproveitando a estrutura e a ementa.
