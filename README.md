# site-compet
Site do PET de Engenharia da Computação do CEFET/MG


## Instalando dependências
```
# pip install web.py
```


## Iniciando a aplicação

```
$ python code.py
```


## Acesso servidor

```
$ ssh compet@compet.decom.cefetmg.br
<senha>
```


## Atualizar repositório

```
$ cd site-compet
$ git pull
```


## Executar servidor

```
$ cd site-compet
$ sudo ./run
```
